# temedica

> Steps for iOS, android works too (Run with Android Studio)
### Steps
* `yarn install`
* `cd ios && pod install`
* `yarn start`

#### Must Have API running https://github.com/p-sebastian/temedica_api

* go into `src/utils/api.util.ts`
* change line 4, into your computers IP address
